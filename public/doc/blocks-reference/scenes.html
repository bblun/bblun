场景 Scenes ​

这类拼图用于加载资产、创建环境、控制场景属性。

场景属性 scene_properties_accessors

设置或获取场景属性。

环境色 ambientColor

定义用于模拟环境色的颜色，默认(0, 0, 0)

背景色 clearColor

定义颜色用于清除渲染缓存

环境纹理 environmentTexture

用于所有PBR材质的反射纹理。因为在大多数情况下它们是共用的，所以在这里设置比在每个材质中分别设置都更方便。也可以在创建环境时设置它。
注意，此处值应是立方体纹理，而不是纹理的地址。

环境强度 environmentIntensity

在所有PBR材质的环境强度。这会使IBL照明整体变暗或增强（反射和漫反射）。由于在大多数场景中，它们是相同的（多房间等除外），因此在此处设置比在所有材质中都更容易。

默认光标 defaultCursor

定义默认的HTML光标，可选值auto default none pointer grab等
详见：https://developer.mozilla.org/zh-CN/docs/Web/CSS/cursor

经过光标 hoverCursor

定义HTML光标用于当经过交互元素时，可选值auto default none pointer grab等
详见：https://developer.mozilla.org/zh-CN/docs/Web/CSS/cursor

重力 gravity

定义场景重力 (仅用于碰撞)

动画时间缩放 animationTimeScale

设置或获取动画播放速度的统一缩放值

强制显示边界盒 forceShowBoundingBoxes

设置是否渲染所有的边界盒

激活相机 activeCameras

场景中所有的激活相机

雾颜色 fogColor

设置雾效颜色

雾强度 fogDensity

设置雾效强度

雾起点 fogStart

设置雾效开始距离

雾终点 fogEnd

设置雾效结束距离

雾模式 fogMode

设置雾效模式
0-无
1-指数
2-指数2
3-线性

启用雾 fogEnabled

设置是否在当前场景中启用雾效

启用灯光 lightsEnabled

设置在当前场景中是否启用灯光 lights

启用阴影 shadowsEnabled

设置在当前场景中是否启用阴影 shadows

启用骨骼 skeletonsEnabled

设置在当前场景中是否启用骨骼 skeletons

启用纹理 texturesEnabled

设置在当前场景中是否启用纹理 textures

启用动画 animationsEnabled

设置是否启用动画

启用音频 audioEnabled

设置是否启用音频支持

启用碰撞 collisionsEnabled

设置是否在当前场景中启用碰撞

启用镜头光晕 lensFlaresEnabled

设置在当前场景中是否启用镜头光晕 lens flares

启用粒子 particlesEnabled

设置在当前场景中是否启用粒子 particles

启用物理 physicsEnabled

设置在当前场景中是否启用物理引擎 physic engines

启用后处理 postProcessesEnabled

设置在当前场景中是否启用后处理 postprocesses

启用灯光探头 probesEnabled

设置在当前场景中是否启用灯光探头 probes

启用精灵 spritesEnabled

设置在当前场景中是否启用精灵 sprites

使用不依赖透明顺序 useOrderIndependentTransparency

设置是否使用与顺序无关的透明度，尽管性能受到影响

使用右手坐标系 useRightHandedSystem

设置应用背景 scene_set_background

注意：使用此拼图时，不能同时使用天空盒背景。

创建地面投射环境 scene_create_projected_ground_environment

创建一个将反射纹理投射到地面的天空盒。
参数：
大小 - 天空盒大小
半径 - 投射环境纹理的半径
高度 - 投射点的高度
反射纹理 - 为PBR材质提供反射的立方体纹理

创建环境 scene_create_default_environment

创建新环境。
需要指定天空盒纹理、地面纹理和环境纹理，若未指定，则从babylon远程服务器加载

根位置 rootPosition

默认的根网络位置，如果“自动大小”为假

自动大小 sizeAuto

自动计算适配场景的最佳元素尺寸

环境纹理 environmentTexture

场景中使用的环境纹理。可以是立方体纹理或者贴图地址，如果未指定纹理或地址，则从babylon远程服务器加载

创建天空盒 createSkybox

是否创建天空盒

天空盒颜色 skyboxColor

默认混入天空纹理的颜色。默认为背景色

天空盒大小 skyboxSize

指定天空盒大小

天空盒纹理 skyboxTexture

用于天空盒的纹理。如果未指定，则从babylon远程服务器加载

背景Y旋转 backgroundYRotation

天空盒背景沿场景的Y轴旋转。这有助于让场景的主光与背景对齐

创建地面 createGround

是否创建地面

地面颜色 groundColor

默认混入地面纹理的颜色
默认为背景色

地面不透明度 groundOpacity

指定地面不透明度

地面大小 groundSize

指定地面大小

地面纹理 groundTexture

用于地面的纹理。如果未指定，则从babylon远程服务器加载

地面Y偏移 groundYBias

指定一个地面垂直方向的偏移值，以确保物体与地面不会重叠

启用地面反射 enableGroundMirror

创建镜面纹理结合到地面

地面反射量 groundMirrorAmount

指定地面镜面可见度

地面反射模糊内核 groundMirrorBlurKernel

指定地面镜面模糊内核大小

地面反射衰减距离 groundMirrorFallOffDistance

指定地面镜面衰减距离。有助于减少反射大小

地面反射菲涅耳权重 groundMirrorFresnelWeight

指定地面镜面反射率权重。使用背景材质的标准权重来设置镜面的菲涅耳效应

地面反射大小比率 groundMirrorSizeRatio

指定地面镜面大小比率
默认0.3因为默认内核为64.

地面反射纹理类型 groundMirrorTextureType

指定地面镜面纹理类型

启用地面阴影 enableGroundShadow

启用地面接收阴影

地面阴影级别 groundShadowLevel

防止地面的阴影死黑

创建图像处理 setupImageProcessing

在场景中创建图片后处理，默认开启

启用色调映射 toneMappingEnabled

指定在场景中是否启用色调映射
默认为真，如果"创建图像处理"为真

镜头对比度 cameraContrast

场景对比度
默认1.6，如果"创建图像处理"为真

镜头曝光度 cameraExposure

场景曝光度
默认0.6，如果"创建图像处理"为真

加载容器 addContainerTask

创建一个加载容器任务。容器可以用来管理一个资产列表（如节点、摄像机、灯光、网格等）。这些资产与场景依然关联，但不被渲染。可以把容器想象成一个实体的水池。

开始&监听加载 scene_listen_loading_event

执行加载任务，并在加载过程中和加载结束时执行对应插槽中的拼图。
通过拼图右键可创建“加载百分比”和“获取加载内容”拼图。
参数：
加载界面 - 为空表示不使用加载界面
当加载时 - 资产加载过程，该插槽可通过齿轮图标在高级设置中开启，配合右键菜单中的加载百分比，可实现加载进度条展示
当完成时 - 资产加载完成，但未渲染
