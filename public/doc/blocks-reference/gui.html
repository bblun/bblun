界面 GUI ​

这类拼图用于加载通过GUI编辑器实现的用户界面，并操作界面中的控制器，实现交互。若使用了这类拼图，在发布时注意勾选GUI模块。

创建全屏UI gui_create_adt_for_fullscreen_ui

创建全屏UI。一个场景中只能有一个全屏UI。
参数：
名字 - UI的名字
视网膜屏幕 - 用于在视网膜屏幕或高ppi（像素每英寸h）屏幕上让UI显示更加清晰
JSON - 包含纹理信息的json数据，可为空

全屏UI属性 gui_adt_property

设置或获取全屏UI属性。

u偏移 uOffset

定义纹理沿UV坐标的u轴的偏移值

u缩放 uScale

定义纹理沿UV坐标的u轴的缩放值

u角度 uAng

定义纹理沿UV坐标的u轴旋转的度数（单位弧度）

u旋转中心 uRotationCenter

定义旋转的中心 (U)

v偏移 vOffset

定义纹理沿UV坐标的v轴的偏移值

v缩放 vScale

定义纹理沿UV坐标的v轴的缩放值

v角度 vAng

定义纹理沿UV坐标的v轴旋转的度数（单位弧度）

v旋转中心 vRotationCenter

定义旋转的中心 (V)

w角度 wAng

定义纹理沿UV坐标的w轴旋转的度数（单位弧度）

w旋转中心 wRotationCenter

定义纹理沿UV坐标的v轴旋转的度数（单位弧度）

平铺U wrapU

贴图平铺方式
0-裁切
1-重复
2-镜像

平铺V wrapV

贴图平铺方式
0-裁切
1-重复
2-镜像

平铺R wrapR

贴图平铺方式
0-裁切
1-重复
2-镜像

UV通道 coordinatesIndex

定义要使用的UV通道，从0开始，默认为0。

映射模式 coordinatesMode

纹理映射方式
0-直接模式
1-球形模式
2-平面模式
3-立方体模式
4-投射模式
5-天空盒模式
6-逆立方体模式
7-等矩圆柱模式
8-固定等矩圆柱模式
9-固定等矩圆柱镜像模式

含Alpha hasAlpha

纹理中是否包含Alpha通道。

级别 level

纹理的强度或力度。

各向异性过滤级别 anisotropicFilteringLevel

需要兼容的硬件和浏览器（支持各向异性过滤），用于定义纹理中各向异性过滤的级别。越高越好，但越慢。默认值为4，兼顾质量和效率。

地址 url

纹理地址。

反转Z invertZ

立方体纹理沿Z轴反转。

旋转Y rotationY

立方体纹理沿Y轴旋转的弧度数。

当销毁时 onDisposeObservable

获取当纹理销毁时事件。

当加载时 onLoadObservable

获取当纹理加载时事件。

当开始渲染时 onBeginRenderObservable

开始渲染事件

当控件选中时 onControlPickedObservable

控件选中事件

禁用拾取 disablePicking

控件是否禁用拾取

每帧检查指针 checkPointerEveryFrame

如果为真，则如果GUI连接到网格或存在连接到网格的控件，则每帧场景渲染都将触发GUI的指针事件。这能捕捉由于相机或网格移动而在GUI周围移动的指针，但会消耗性能。

背景 background

背景颜色

创建网格纹理 create_adt_for_mesh_texture

以投射模式创建一个新的高级动态纹理（附着在网格上），但不会为网格创建一个新的材质。需手动连接该纹理。
网格 - 要应用纹理的网格
宽度 - 纹理宽度
高度 - 纹理高度
指针移动 - 是否纹理必须捕获指针移动事件
JSON - 包含纹理信息的json数据，可为空

设置UI图层蒙板 set_ui_layer_mask

设置图层蒙板是为了将图层限制在特定相机下。如果图层和相机蒙板值相机，那么图层可在此相机中渲染出来。

添加控件 gui_add_control

向画布纹理(全屏UI)或容器控件中添加子控件。目标为空表示添加到画布纹理。
子控件可以是一个控件或控件数组。

设置网格定义 gui_set_grid_definition

设置网格的行和列的定义。
参数：
网格-要设置的网格
类型-行高或列宽
索引-行或列的索引，从0开始
值-要设置的行或列的值
单位-像素或比例(0-1)

获取控件子级 gui_get_child_controls

返回此控件的所有子级控件

离散控件 gui_move_to_non_overlapped_position

根据控件的重叠组属性将控件移动到不重叠的位置。仅支持全屏UI。
参数：
重叠组-要离散控件的重叠组，为0时表示离散所有控件
速度-控件将以多快的速度移动到不重叠的位置
斥力-控件之间的排斥力度

移动控件 gui_move_control_to_vector

移动控件到一个映射到屏幕上的三维坐标点位置。仅支持全屏UI。

连接控件 gui_link_control_with_mesh

将控件与网格连接到一起。仅支持全屏UI。
参数：
控件 - 要连接的控件
目标 - 网格或变换节点
偏移X - 控件与连接网格在X轴的偏移值(单位：像素)
偏移Y - 控件与连接网格在Y轴的偏移值(单位：像素)

绘制连接线 gui_draw_line_from_mesh_to_control

从网格到控件绘制一条直线。仅支持全屏UI。
参数：
名字 - 直线控件的名称
网格 - 要连接的网格或变换节点
控件 - 要连接的控件
选项 - 控制直接的参数
