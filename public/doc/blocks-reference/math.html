数学 Math ​

这类拼图用于提供数字、向量等数学方法。

数字 math_number

用于提供各种数字。支持整数、浮点数，及科学计数法。快捷键：Shift + N

角度 math_angle

用于方便的输入角度（单位：度）。

16进制 math_hex

数字的16进制表示。

转为数字 math_convert_to_number

用于将任何类型的值转为数字。

转为弧度 math_convert_to_radians

将一个角度值转为弧度值。

算术运算 math_arithmetic

用于执行基本的算术运算：加法（+）减法（-）乘法（*）除法（/），以及指数运算 (^)。

平方根 math_single

用于执行以下数学运算：平方根（√）、绝对值又称模数（|x|）、负（-）、自然对数（ln）、log10、自然指数（e^）和十的乘方（10^）。
注意，浮点运算并不总是100%准确，在比较数值时可能会产生不可预测的结果。为了消除这些精度问题，可以使用带精度的四舍五入。

三角函数 math_trig

计算三角函数正弦（sin）、余弦（cos）和正切（tan）的值，以及它们的反对应的反正弦（asin）、反余弦（acos）和反正切（atan）的值。拼图需要（或返回）以度表示的值。

常数 math_constant

返回以下数学常数：派（π）、欧拉数e、黄金比率（φ）、2的平方根、1/2的平方根和无穷大值（∞）。

数字属性 math_number_property

检查数字是偶数、奇数、素数、整数、正数、负数，还是可以被其他数字整除。如果是，返回真；否则返回假。

舍入 math_round

用于将一个数四舍五入。
参数：
舍入round - 将一个数字舍入到最接近的整数：2.49四舍五入值为2，2.5四舍五入值为3。
向上舍入round up - 将数字向上舍入到最接近的整数：2.1四舍五入值为3。
向下舍入round down - 将数字向下舍入到最接近的整数：2.9四舍五入值为2。

舍入到位数 math_round_to_decimal

用于将一个数四舍五入到指定小数点位数。

计算数组 math_on_list

对列表里的数字执行指定操作。
参数：
和sum - 计算列表中所有数字的和。
最小值min - 查找最小的数字。
最大值max - 查找最大的数字。
平均值average - 查找算术平均值（所有数字的总和除以列表的长度）。
中间值median - 查找中间值（当所有数字从最小到最大排序时作为中间数）。
常见值modes - 查找模（作为列表中出现频率最高的数字列表）。
标准差standard deviation - 计算标准偏差（作为算术平均值的平方偏差平均值的平方根）。
随机元素random item - 返回随机选择的项。

求余 math_modulo

返回一个数除以另一个数之后的余数（也称为模运算）。

限制 math_constrain

将数字限制在指定范围内。

随机整数 math_random_int

返回指定数字之间的随机整数（包含最大最小值）。最大最小值可以不是整数，因为这个拼图默认会执行四舍五入。

随机浮点数 math_random_float

返回一个介于0（包含）到1（排除）之间的随机浮点数。

反正切2 math_atan2

返回以弧度表示的 y/x 的反正切值。计算时atan2 比 atan 稳定。

混合数字 math_mix

返回数字A和B通过系数alpha混合的插值。混合系数的范围是0-1。

数字重映射 math_remap

将数字从一个取值范围映射到另一个取值范围。

向量2 math_vector2

二维向量

向量3 math_vector3

三维向量是3D几何中使用的主要对象。它可以表示空间中一个点的坐标，或者一个方向。

向量4 math_vector4

用于欧拉转四元数

克隆向量 math_clone_vector

从当前三维向量复制一个新的三维向量

向量3运算 math_arithmetic_vector

向量的运算。

加 add

vector3 返回一个新的三维向量，结果是两个三维向量的和

减 subtract

vector3 返回一个新的三维向量，结果是两个三维向量相减

乘 multiply

vector3 返回一个新的三维向量，结果是两个三维向量的数量积

除 divide

vector3 返回一个新的三维向量，结果是第一个三维向量除以第二个三维向量

负 negate

vector3 返回一个与当前三维向量相反的三维向量

归一化 normalize

vector3 归一化当前三维向量。注意，这是一个原地操作

长度 length

number 返回三维向量的长度

缩放 scale

vector3 返回一个新的三维向量，其中当前三维向量坐标乘以浮点"scale"

中点 center

vector3 返回一个新的三维向量，位置是两个三维向量的中点

距离 distance

number 返回两个三维向量的距离

点乘 dot

number 返回两个三维向量的点乘（内积）

叉乘 cross

vector3 返回一个新的三维向量，结果是两个三维向量和叉乘（外积）。叉乘与两个三维向量正交。
