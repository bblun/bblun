灯光 Lights ​

这类拼图用于创建灯光和阴影。

创建灯光 create_light

创建灯光。

点光源 PointLight

在场景中创建一个点光源。点光源会向每个方向发光，可以产生阴影。

亮度 intensity

灯光的亮度。

漫反射 diffuse

灯光的基础颜色

高光 specular

灯光的高光颜色
注意: 不会影响PBR材质

位置 position

灯光在场景中的位置

父级 parent

设置或获取节点的父级

启用阴影 shadowEnabled

设置是否为该光源启用阴影。这可以帮助关闭/打开阴影,而不需要解除当前的阴影生成器

阴影最小Z shadowMinZ

阴影投射裁切最小z值

阴影最大Z shadowMaxZ

阴影投射裁切最大z值

亮度模式 intensityMode

设置灯光亮度的衰减模式。它只与PBR材质相关

灯光贴图模式 lightmapMode

设置该光源的灯光贴图模式
可选值:
0 - 默认
1 - 仅高光
2 - 仅阴影

半径 radius

用于在PBR材质中模拟柔和的面光。
光滑的材质表面会把灯光反射成一个锐利的点，此参数可以使灯光边缘柔和。

排除网格 excludedMeshes

设置不受该灯光影响的网格

平行光 DirectionalLight

在场景中创建一盏平行光。平行光会从任何位置沿指定方向发光，可以产生阴影。

亮度 intensity

灯光的亮度。

漫反射 diffuse

灯光的基础颜色

高光 specular

灯光的高光颜色
注意: 不会影响PBR材质

位置 position

灯光在场景中的位置

方向 direction

灯光的方向

父级 parent

设置或获取节点的父级

自动计算阴影Z边界 autoCalcShadowZBounds

自动计算投影矩阵的"阴影最小Z"和"阴影最大Z"，以在每一帧上达到最佳拟合（包括所有阴影投射网格）。"自动更新扩展"必须设置为真才能正常工作

自动更新扩展 autoUpdateExtends

自动计算投影矩阵的"阴影最小Z"和"阴影最大Z"，以在每一帧上达到最佳拟合（包括所有阴影投射网格）。

聚光灯 SpotLight

在场景中创建一盏聚光灯。聚光灯是类似锥形的灯光，可以产生阴影。
主要属性：
位置 - 灯光在场景中的位置
方向 - 灯光在场景中的方向
角度 - 灯光锥角度数
指数 - 灯光距离光源的衰减速度

亮度 intensity

灯光的亮度。

漫反射 diffuse

灯光的基础颜色

高光 specular

注意: 不会影响PBR材质

位置 position

方向 direction

灯光的方向

锥角 angle

聚光灯锥角的弧度数

内锥角 innerAngle

聚光灯圆锥体内角角度衰减，仅在gltf衰减模式中使用

指数 exponent

灯光衰减随距离而变化的速度

父级 parent

半球光 HemisphericLight

在场景中创建一盏半球光。半球光模拟的是环境光，它的方向属性是灯光的反射方向，而不是来光方向。半球光不能产生阴影。

亮度 intensity

灯光的亮度。

漫反射 diffuse

灯光的基础颜色

高光 specular

灯光的高光颜色
注意: 不会影响PBR材质

底光 groundColor

底光是与灯光方向相反的灯光。可以将漫反射和高光视为从光源处发出的光，而底光是从相反方向发出的光。

方向 direction

灯光的方向

父级 parent

设置或获取节点的父级

灯光属性 light_property

设置或获取灯光属性。

设置灯光目标 set_light_target

将灯光的方向设置为指定的目标点。

获取灯光 get_light_by_name

根据名称获取场景中的灯光。

获取阴影 get_shadow_from_light

获取灯光阴影。

销毁灯光 dispose_light

销毁灯光，并释放相关资源。

创建阴影 create_shadow_for_light

创建阴影。
参数：
灯光：要产生阴影的灯光，平行光、聚光灯、点光源中的一种
选项：阴影选项，通过右键菜单-创建选项，创建选项拼图

阴影 ShadowGenerator

阴影生成器，支持平行光、点光、聚光灯

偏移 bias

设置偏移: 偏移用于深度防止阴影错误 (在灯光方向上).

法线偏移 normalBias

设置法线偏移：偏移用于深度防止阴影错误(沿法线方向并与灯光/法线角度成比例)

暗度 darkness

设置或获取实际阴影暗度

贴图大小 mapSize

获取或设置存储阴影的纹理的大小

透明阴影 transparencyShadow

获取或设置是否开启透明阴影

模糊框偏移 blurBoxOffset

设置模糊方框偏移：偏移用于模糊通道
仅当使用内核模糊为false时有效

模糊缩放 blurScale

设置模糊缩放：与主阴影贴图相比,模糊纹理的比例
2表示半尺寸

深度缩放 depthScale

在ESM模式下设置深度比例。这会覆盖存储在灯光中的缩放

过滤模式 filter

设置或获取当前阴影生成器模式。返回值是一个数字，它代表阴影贴图中的一个模式。
0 - NONE
1 - Exponential
2 - Poisson
3 - Blurred exponential
4 - Close exponential
5 - Blurred close exponential
6 - PCF
7 - PCSS

过滤质量 filteringQuality

设置PCF或PCSS质量。
0-高质量
1-中质量
2-低质量

半影比率 contactHardeningLightSizeUVRatio

此参数影响阴影边缘虚化效果，在PCSS中有效。

使用内核模糊 useKernelBlur

设置模糊通道是内核模糊(如果为真)还是方框模糊。仅在已过滤模式（使用模糊指数阴影贴图等）下有效

模糊内核 blurKernel

设置模糊内核：模糊通道内核大小
仅当使用内核模糊为true时有效

级联阴影 CascadedShadowGenerator

级联阴影生成器，仅支持平行光，用于模拟大型室外场景的太阳光阴影。

级联数量 numCascades

用于CSM的级联数量1-4

稳定级联 stabilizeCascades

如果希望阴影的边缘在旋转相机时不会“游泳”/“闪闪发光”，可设置此参数为真。要权衡的是，启用此设置时，阴影渲染会失去一些精度。

lambda lambda

设置lambda参数。此参数用于分割相机视锥体并创建级联。它是一个介于0-1之间的值。如果为0，视锥体的w分割为均匀分割，如果为1，为对数分割。介于两者之间的值，它是均匀分割算法和对数分割算法的线性组合。

级联混合比例 cascadeBlendPercentage

获取或设置两个级联之间的混合百分比（值介于0-1）。默认为0.1（10%混合）。

深度钳制 depthClamp

启用后，它可以提高阴影质量，因为不需要调整视锥体的近Z平面来考虑远处的阴影投射网。
请注意，此属性与PCSS过滤模式不兼容。

自动计算深度边界 autoCalcDepthBounds

启用后，它可以大大提高阴影质量，但代价是需要更多的GPU工作。使用此选项时，应该增加lambda参数的值，甚至将其设置为1以获得最佳结果。

阴影最大Z shadowMaxZ

阴影最大Z距离。这是不显示阴影的极限。默认为camera.maxZ

偏移 bias

设置偏移: 偏移用于深度防止阴影错误 (在灯光方向上).

法线偏移 normalBias
