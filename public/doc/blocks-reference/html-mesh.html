创建HTML网格 create_html_mesh

用于将HTML内容嵌入到3D场景中，可以嵌入视频、网站、PDF或任何HTML元素。
注意，有的远程资源是不允许跨域访问的，这些资源无法用于此拼图。

类型 type

要嵌入场景的HTML内容的类型。

HTML元素 htmlElement

HTMLElement。

宽 width

网格宽度。注意，网格大小在这里指定后不能再缩放，可以移动和旋转。

高 height

网格高度。注意，网格大小在这里指定后不能再缩放，可以移动和旋转。

捕获指针 pointerEnter

当指针移入时自动捕获指针，与嵌入的HTML内容交互，此时无法与场景交互。
