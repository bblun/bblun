相机 Cameras ​

创建相机 create_camera

创建漫游相机、环绕相机、跟随相机等。

漫游相机 UniversalCamera

漫游相机。可用于第一人称射击游戏，支持键盘、鼠标、触屏和游戏手柄。
名字: 定义场景中相机的名称
位置: 定义相机在场景中的初始位置

最小Z minZ

定义相机可以观察的最小距离

最大Z maxZ

定义相机可以观察的最大距离

惯性 inertia

定义相机默认惯性

图层蒙板 layerMask

数字）限制相机只有查看有相同图层蒙板的物体。

模式 mode

0-透视相机
1-正交相机

FOV fov

视野大小，单位弧度

目标 target

定义相机目标点

锁定目标 lockedTarget

定义相机注视的目标对象或位置。注意，锁定目标将禁用相机平移。

位置 position

定义相机本地空间位置。

旋转 rotation

定义相机的旋转。

转向敏感度 angularSensibility

设置或获取鼠标转动的敏感度 (默认 2000.0) 值越高敏感度越低

触屏转向敏感度 touchAngularSensibility

定义相机转向的触摸敏感度，值越高越快

触屏移动敏感度 touchMoveSensibility

定义相机移动的触摸敏感度，值越高越快

速度 speed

定义当前相机的速度

检测碰撞 checkCollisions

开启或关闭相机与场景中其它物体的碰撞

应用重力 applyGravity

开启或关闭相机重力属性

椭球体 ellipsoid

定义相机的椭球体碰撞体。用于模拟相机实体，如玩家的身体

椭球体偏移 ellipsoidOffset

为椭球体在相机周围的位置定义一个偏移量。可用于将身体的中心点设在重心附近，而不是它的头部

前进键 keysUp

设置或获取用于控制相机向前移动的键盘按键码数组
查看按键码通过下拉菜单-工具-按键码

后退键 keysDown

设置或获取用于控制相机向后方移动的键盘按键码数组
查看按键码通过下拉菜单-工具-按键码

左移键 keysLeft

设置或获取用于控制相机向左移动的键盘按键码数组
查看按键码通过下拉菜单-工具-按键码

右移键 keysRight

设置或获取用于控制相机向右移动的键盘按键码数组
查看按键码通过下拉菜单-工具-按键码

左转键 keysRotateLeft

设置或获取用于控制相机向左旋转的键盘按键码数组
查看按键码通过下拉菜单-工具-按键码

右转键 keysRotateRight

设置或获取用于控制相机向右旋转的键盘按键码数组
查看按键码通过下拉菜单-工具-按键码

上移键 keysUpward

设置或获取用于控制相机向上移动的键盘按键码数组
查看按键码通过下拉菜单-工具-按键码

下移键 keysDownward

设置或获取用于控制相机向下方移动的键盘按键码数组
查看按键码通过下拉菜单-工具-按键码

环绕相机 ArcRotateCamera

环绕相机。该相机可围绕前方的目标位置进行旋转（就像卫星绕行地球）。可通过鼠标或触摸事件控制。
名字: 相机名称
alpha: 定义相机围绕纵轴转动的度数（弧度）
beta: 定义相机围绕横轴转动的度数（弧度）
半径: 定义相机与目标点的位置
目标: 定义相机的目标点

最小Z minZ

定义相机可以观察的最小距离

最大Z maxZ

定义相机可以观察的最大距离

惯性 inertia

定义相机默认惯性

图层蒙板 layerMask

数字）限制相机只有查看有相同图层蒙板的物体。

模式 mode

0-透视相机
1-正交相机

FOV fov

视野大小，单位弧度

位置 position

定义相机位置

目标 target

定义相机目标点

锁定目标 lockedTarget

定义相机注视的目标对象或位置。注意，锁定目标将禁用相机平移。

目标屏幕偏移 targetScreenOffset

定义相机目标点在屏幕上的偏移

alpha alpha

定义相机围绕纵轴转动的度数（弧度）

beta beta

定义相机围绕横轴转动的度数（弧度）

半径 radius

相机与目标点的距离

转向敏感度X angularSensibilityX

获取或设置指针沿X轴的角度敏感度，或摄像机旋转的速度

转向敏感度Y angularSensibilityY

获取或设置指针沿Y轴的角度敏感度，或摄像机旋转的速度

平移敏感度 panningSensibility

定义指针平移敏感度或相机移动速度

捏合百分比 pinchDeltaPercentage

设置或获取指针缩放增量百分比或相机缩放的速度。它定义了使用缩放时用作增量的相机半径的百分比。

滚轮速度百分比 wheelDeltaPercentage

设置或获取鼠标滚轮速度百分比或相机缩放的速度。它定义了在使用捏合变焦时，将当前相机.半径的百分比作为delta。

速度 speed

定义当前相机的速度

检测碰撞 checkCollisions

开启或关闭相机与场景中其它物体的碰撞

碰撞半径 collisionRadius

定义相机的碰撞半径。该参数模拟了一个围绕相机的球体。

alpha下限 lowerAlphaLimit

水平轨道最小弧度
可用于限制相机在场景中的移动

alpha上限 upperAlphaLimit

水平轨道最大弧度
可用于限制相机在场景中的移动

beta下限 lowerBetaLimit

垂直轨道最小弧度
可用于限制相机在场景中的移动

beta上限 upperBetaLimit

垂直轨道最大弧度
可用于限制相机在场景中的移动

半径下限 lowerRadiusLimit

相机距离目标的最小距离
可用于限制相机在场景中的移动

半径上限 upperRadiusLimit

相机距离目标的最大距离
可用于限制相机在场景中的移动

相机属性 camera_property

设置或获取相机属性。

激活相机 set_get_active_camera

设置或获取激活相机

启用禁用相机 enable_disable_camera_control

启用或禁用相机控制器。

碰撞移动相机 move_camera_with_collison

相机在场景中移动时，会检测碰撞并避免穿透物体。

获取相机方向 get_camera_direction

获取相机相对于指定局部轴的方向。
轴：定义参考轴以提供一个相对的方向。

获取相机射线 get_camera_forward_ray

获取相机正前方射线
长度: 射线长度

销毁相机 dispose_camera

销毁相机及相关资源

自动环绕 autorotate_camera

在没有用户交互时，为环绕相机创建匀速自动旋转效果
速度: (number) 相机围绕模型旋转的速度
等待时间 (number) 在用户交互之后相机旋转之前的等待时间（单位秒）
加速时间: (number) 相机从静止到正常旋转的加速时间（单位秒）

自动回弹 bounce_camera

在达到最小和最大半径时，为环绕相机创建回弹效果
时长: (number) 动画时长，单位秒
最小距离: (number) 相机与目标的最小距离 (相机无法继续靠近).
最大距离: (number) 相机与目标的最大距离 (相机无法继续靠近).

自动注视 zoom_camera

当环绕相机的目标设置到一个网格时，可以自动定位相机。也可用于阻止相机穿透地面。
时长: (number) 动画时长，单位秒
